The PID controller consists of three components:

  • Proportional (P): Reduces the error by applying a correction proportional to the deviation.
  • Integral (I): Eliminates steady-state error by integrating past errors.
  • Derivative (D): Reduces overshoot by predicting future errors.

ğŸ”¢ PID Control Algorithm (Arduino Code)

double Kp = 2.0, Ki = 1.0, Kd = 0.5;
double previous_error = 0, integral = 0;
void PID_Control(double setpoint, double processVariable) {
    double error = setpoint - processVariable;
    integral += error * dt;
    double derivative = (error - previous_error) / dt;
    double output = Kp * error + Ki * integral + Kd * derivative;
    previous_error = error;
    analogWrite(PWM_pin, constrain(output, 0, 255));
}
