Overview

The **Linear Position Control System** controls **linear movement** using a DC motor, rotary encoder, and PID controller. It allows precise positioning with **manual and automatic control modes**.

double Kp = 1.2;
double Ki = 0.7;
double Kd = 0.3;
double setPoint = 50; // Target position in mm
double processVariable = 0;
double controlOutput = 0;

void loop() {
    double error = setPoint - processVariable;
    double integral = integral + error;
    double derivative = error - previous_error;
    controlOutput = Kp * error + Ki * integral + Kd * derivative;
    previous_error = error;
}
