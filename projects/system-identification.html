Overview

This project focuses on real-time system identification of a DC motor using MATLAB, Simulink, and Arduino. The goal is to model the motor dynamics, estimate a transfer function, and implement closed-loop control.

📌 Hardware & Software Used

  • 🔹 Arduino MEGA 2560
  • 🔹 DC Motor with Quadrature Encoder
  • 🔹 L298N Motor Driver
  • 🔹 MATLAB & Simulink
  • 🔹 System Identification Toolbox

⚙️ Technical Explanation

We first apply a voltage input to the motor and measure the angular velocity using an encoder. The collected input-output data is used in MATLAB’s System Identification Toolbox to estimate a transfer function.

📊 Transfer Function Model

The estimated model for the system is:

H(s) = 6.853 / (s² + 17.72s + 104.8)
